# Cyber-Resilience Framework for SMEs in Senegal
This project explores cybersecurity resilience strategies for small and medium-sized enterprises (SMEs) in Senegal, with a focus on risk management, vulnerability assessment, and incident response planning.
Developed as part of a master’s research project in cybersecurity at ECPI Dakar, the initiative examines how structured security frameworks and policy-oriented approaches can strengthen organizational resilience and contribute to broader economic and digital stability in developing economies.
